Modifying Point Feature

Data from a PostgreSQL server, is transformed into a point feature on the fly and displayed on the web map. Predefined legends are applied automatically. You can search for any feature, change its location, update the related attributes, and can delete it. All these changes are in real time.

Edit options are limited for selected users only. Generally, an admin manages the access previleges.

Modifying Line/Polyline Feature

You can draw new line, change the alignment of any existing line, and update the attributes of the selected feature. These changes are reflected in the server, which means that changes made are in real time. Color and thickness of lines change on its own with the changes of the associated attributes.

Locations can be searched on map, and routing is also possible with an API token.

Modifying Polygon Feature

You can digitize buildings right from the satellite map and put them in your database. And your digitized shapes are immediately available to other people of your organization. After that, they can add to other attributes, if required. It enables a real time collective working environment.

You can also do the digitization and editing in any GIS software, and view the outputs immediately with predefined legends in the web map.

Performing SQL queries on WebMap

Unlock the power of data with a single SQL query, as your web map transforms complex information into an intuitive visual representation. You can dive into the structure of your spatial data, and can display any kind of complex filtering.

Nonetheless, be aware of the SQL Injection. Proper measure must be taken to save the database from malicious SQL commands.

Downloading Spatial Data from WebMap

GeoJSON provides a lightweight and flexible way to store and transmit spatial data, making it widely used in web-based mapping applications and GIS platforms. GeoJSON files for all the spatial data can be directly downloaded from my WebGIS portal. Also they are available in excel files.

You can open the GeoJSON files in QGIS and export those to shapefile, if it is required.

Generating Report from WebGIS Server

It is good to have a real time reporting system on your WebGIS software. It contains the update-to-date summary of all the features of your project. If any user modifies the attribute of any asset, it will be automatically reflected in the reporting system.

You can modify the reporting tables according the need of your organization.
