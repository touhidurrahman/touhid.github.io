Hello, I'm Md. Touhidur Rahman, a seasoned GIS professional with over 5 years of hands-on experience.

My expertise lies in developing customized WebGIS systems and writing complex scripts for geospatial analysis. Based on the project needs, I utilized multiple tools and technologies to achieve optimum output.

WebGIS Developer

As a WebGIS Developer, I like to use open source technologies to develop WebGIS systems that are capable of handling real-time data from multiple users and APIs.

Following tools are my daily drivers:

  • For mapping library, I use Leaflet.
  • As a frontend language, I use JavaScript and jQuery.
  • I use both Python and PHP as backend languages. Based on purpose, I frequently switch between these two.
  • AJAX for communicating between frontend and backend.
  • For database, I prefer PostgreSQL.
  • I extensively use PostGIS for doing spatial analysis and query.
  • Like to form complex and efficient SQL queries for faster data processing.
  • Proficiency in the Esri ecosystem alongside my dedication to open-source technologies.
    I use different techniques to provide security for the WebGIS portals that I develop. These includes protecting against: SQL Injection, Cross-site Scripting, DDoS, Phising, and MitM Attack.

GIS Programmer

I automate complex and time consuming geospatial tasks with Python. Many time the scripts that I write, take hours to process large amount data. But it saves months of work without any or less error.

  • I often use ArcPy, if I have to borrow any functionality from Esri.
  • GeoPandas is my favorite. It is fast, robust and reliable.
  • For manipulating geometry and making new ones on the fly, I use Shapely.
  • For making interactive maps with python, i turn to Folium.
  • For visualization and dashboards, I use Plotly.
    First, I write the scripts in Jupyter Notebook and test it. Finally I turn them into a single script for users.
